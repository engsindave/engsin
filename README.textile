h1. Enki

This template is a barebones scaffold for anyone who wants a blank slate for a super clean website with a blog. It includes all of the features from the original Enki Blog along with some new aesthetic and functional improvements made by myself.

h2. Features

Bootstrap styling for both the front end and the admin side
Responsive navigation bar
Independent homepage from the blog index
Page URLs without the ‘page/’ precursor
Customizable page order in the navigation bar
Improved in-page SEO
PostgreSQL production database for immediate deployment on Heroku
Sticky footer
